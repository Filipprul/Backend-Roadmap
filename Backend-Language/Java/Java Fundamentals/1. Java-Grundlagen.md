# Was ist Java überhaupt?
Java ist eine Programmiersprache und gleichzeitig eine Plattform, die 1995 von Sun Microsystem (heute gehört sie zu Oracle) entwickelt wurde.
Das bedeutet:
- Du schreibst mit Java Programme (die Sprache).
- und du führst sie auf der sogenannten Java Virtual Machine (JVM)aus (die Plattform).

Java wurde so entwickelt, dass Programme auf allen Geräten laufen können, ohne dass du sie neu schreiben musst.
Das war der berühmte Grundsatz:
"Write once, run anywhere" (WORA).

# Eigenschaften von Java
- General-purpose (Allzwecksprache) → Du kannst Java für fast alles verwenden: Spiele, Web, Mobile, Desktop, KI, Server usw.
- Class-based → In Java ist _jede_ Anwendung in Klassen organisiert – das ist der Grundbaustein.
- Object-oriented (objektorientiert) → Alles basiert auf Objekten (Instanzen von Klassen) – sie enthalten Daten (_Attribute_) und Verhalten (_Methoden_).
- Platform-independent → Java-Code läuft auf allen Betriebssystemen dank der JVM.
- Secure & Robust → Java hat integrierte Sicherheitsmechanismen und prüft Code stark zur Laufzeit.
- Fast & Reliable → Durch die Just-In-Time (JIT)-Kompilierung wird der Code zur Laufzeit optimiert.

# Wie funktioniert Java technisch?
1. Du schreibst **Java-Code** → Datei endet mit `.java`
   Beispiel:
    public class Main {
	    public static void main(String[] args) {
		    System.out.println("Hallo Welt!");
	    }
    }

2. Der **Java-Compiler (`javac`)** übersetzt diesen Code in **Bytecode** → Datei `.class`.
3. Die Java Virtual Machine (JVM) liest diesen Bytecode und führt ihn auf _jedem Gerät_ aus, das eine JVM installiert hat (Windows, Linux, Android usw.).

Vorteil:
→ Du musst dein Programm **nicht** an jedes Betriebssystem anpassen.

# Wofür wird Java verwendet?
Java ist sehr vielseitig und wird in vielen Bereichen genutzt:
 Webentwicklung (Backend) → z. B. Spring Boot
- Mobile Apps → Android basiert ursprünglich auf Java
- Spieleentwicklung → z. B. Minecraft
- Unternehmenssoftware → Banken, Versicherungen, ERP-Systeme
- Cloud- & Serveranwendungen
- Künstliche Intelligenz und Datenverarbeitung

# Warum ist Java so beliebt?
- Riesige Community und viele Bibliotheken
- Sehr gute Dokumentation
- Stabil, langfristig und rückwärtskompatibel
- Wird in Schulen, Universitäten und Unternehmen als Standardsprache gelehrt
